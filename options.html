Custom TaxonWorks bangs (!)

Paste any TaxonWorks URL from your browser — the host (and port) is stripped automatically, so the alias works against whichever instance @host points to at query time. Dev URLs like http://localhost:9999/projects/3/select work too. You can also paste just a host-relative path (e.g. /hub), and {} in the path is substituted with the typed query. Don't include the leading ! in the alias field.
Alias Type TaxonWorks URL or path Label

Custom external bangs (~)

Add or override aliases that point to an external site. Enter a full URL template containing {} as the placeholder for the typed query (e.g. https://example.com/search?q={}). External bangs ignore @host. Don't include the leading ~ in the alias field.

Default tab behavior

Where to open results when you press plain Enter (no modifier key and no trailing tab marker in the query). Trailing markers (\ \\ | || \| |\) and modifier-key gestures (Alt+Enter, etc.) always override this.


Backup & sync

Export downloads your settings (custom bangs, host list, default tab behavior) as a JSON file you can save or share. Import reads one back. Use Firefox Sync moves your settings into the synced storage area so they propagate across the Firefox profiles you're signed in to. Off by default — no Mozilla account needed.
